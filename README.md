# SolitonKit MK III Final v2 

![Soliton Radar](https://media4.giphy.com/media/X5HXRzHA78Og7Fr64T/giphy.gif)

**Comprehensive Python SEO Toolkit** - Keyword research, on-page audits, PageSpeed monitoring, density analysis, JSON-LD schema generation.

## 🚀 Features

- **Keyword Extractor** - Serper.dev API pulls SERP keywords (PAA, related, organic titles, snippets)
- **On-Page SEO Crawler** - BeautifulSoup audits meta, schema, hreflang, images, links, readability
- **PageSpeed Insights** - Google PSI v5 Core Web Vitals (LCP/CLS/TBT) + optimization opps
- **Keyword Density** - Smart n-gram (1-3) analysis with SERP competitor benchmarking
- **Schema Generator** - 15+ schema types (Article, Product, FAQ, LocalBusiness, etc.)

## 🛠 Tech Stack

```bash
Python 3.8+ | Tkinter GUI | requests | BeautifulSoup4 | lxml | Pillow
APIs: Serper.dev (2.5k/mo free) | Google PageSpeed (25k/day free)
```

## 📦 Quick Install

```bash
pip install requests beautifulsoup4 lxml pillow
python solitonkit.py
```
